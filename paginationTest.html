Pagination Rules and Guidance

Below is a subset of rules from US Web Design System. For additional guidance see, https://designsystem.digital.gov/components/pagination/.

  • Currently only covers bounded set scenarios.
  • The current page is denoted by a "*" in each expected and actual output.
  • Display at most, 9 elements which includes Previous, Next, Page Numbers, and Overflow indicators (...)
  • If 7 or less pages exist, truncate page number list to equal the number of pages present
  • Always display first, last, current, next, and previous page numbers (e.g. [1][...][7][8*][9][...][22])
